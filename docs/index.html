From Zero to Auto Pilot:

Exploring Kubernetes Operator Capability Levels

What is the L5 operator &

Why was it created?

- The L5 operator is an operator that manages and consumes a flask app with a postgres database utilizing the Crunchy Postgres Operator.
- Following the Operator Framework, the L5 operator was created to demonstrate the capabilities of an operator from Level 1:basic installation to Level 5:autopilot.

Operator Framework

- Operator Framework is an open source toolkit used to manage Operators, in an effective, automated, and scalable way.
- Operators come in different maturity levels so this capability model aims to provide a guide as to what features users can expect from an Operator.
- The operator framework defines 5 levels of operator capabilities. level 1: basic install, level 2: seamless upgrade, level 3: backup & restore, level 4: deep insights, and level 5: autopilot.
- Currently on operatorhub.io 70% of the published operators have level 2 capabilities and only 5% of operators have level 5 capabilities.
This presentation aims to enable participants to create operators with greater capabilities as well as add more capabilities to existing operators.
